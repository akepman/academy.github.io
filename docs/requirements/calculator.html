Калькулятор системных требований

Тип приложения

Не влияет на требования к ОС других серверов (БД, сессий, и т. д.), а также на требования к клиентским компьютерам.

Сервер приложений может быть развернут на Windows с использованием IIS.

Сервер приложений может работать как на Windows, так и на Linux. Для продуктивной среды рекомендуется развертывание на Linux.

База данных

Укажите количество пользователей, которые будут активно работать в системе. Это основной критерий для расчета аппаратных требований.
Схема развертывания

Схема развертывания с отказоустойчивостью предусматривает дополнительные компоненты, такие как балансировщик нагрузки и дополнительные серверы.
Режим восстановления

Режим полного восстановления требует больше места на жестком диске сервера.
Система контроля версий

Если сайт планируется использовать как выделенную среду разработки, то необходимо развернуть хранилище контроля версий, предназначенное для управления изменениями. Разработка и настройка системы при помощи low-code инструментов доступны и на сайтах без хранилища контроля версий.
Портал
Контейнерные компоненты

Сервисы, необходимые для полноценной работы приложения — поиска данных, работы с почтой, дедупликации, моделей машинного обучения. Для установки потребуется отдельный сервер контейнерной виртуализации.
Список контейнерных компонентов

Интегрирует Creatio с поисковой системой ElasticSearch. Используется для поиска данных в Creatio из командной строки.

Необходим для поиска и объединения дублей в разделах Creatio. Не используется без компонента глобального поиска.

Используется для работы инструментов ИИ – прогнозирования значений в полях, формирования рейтинга записей, списков рекомендаций и т. п.

Интегрирует Creatio с почтовыми сервисами MS Exchange и IMAP/SMTP. Используется для работы с почтой.

CREATIO ON-SITE

Укажите условия развертывания Creatio и нажмите "Далее". Калькулятор выполнит расчет системных требований для вашей конфигурации.
