Remarks

По контракту должен возвращать ((EntityLazyProxy)resource).Entity.Value, но возможно стоит возвращать прокси, что бы преждевременно не делать запрос в БД. В таком случае все части кода используещие возвращаемый объект должны знать о EntityLazyProxy. Места где используется: EntityDataServiceQueryProvider.GetResourceType() Вызывается после вызова SaveChanges для сериализации ресурса для методов POST
