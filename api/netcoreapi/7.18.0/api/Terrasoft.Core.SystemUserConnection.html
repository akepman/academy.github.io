Remarks

Так как в приложении хранится и используется один экземпляр SystemUserConnection, это приводит к проблемам, когда параллельные web запросы используют этот экземпляр для выполнения операций с базой данных. Подробности описаны в статье http://tswiki/pages/viewpage.action?pageId=59671630
